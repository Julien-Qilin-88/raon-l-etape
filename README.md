# RAON L'ETAPE

![RAON](img/Raon1.jpg)

**Raon-l'Étape** est une commune française située dans le département des Vosges, en Lorraine, dans la région administrative Grand Est.

Ses habitants sont appelés les *Raonnais(es)*.
